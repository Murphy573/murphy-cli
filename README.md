# murphy-cli

简单的模板生成 cli

### 安装

```bash
$ npm install -g murphy-cli
```

### 使用

```bash
$ murphy create <project-name>
```

### 示例

```bash
$ murphy create my-project
```

**该命令将在 my-project 目录下生成对应项目模板**

### 您可能需要修改

- 打开 `package.json`，按需修改 `name`、`version`、`description` 等字段

- 按需修改 `.env.*` 环境配置文件

- 按需修改 `vue.config.js` 配置文件

- 执行 `npm i`
